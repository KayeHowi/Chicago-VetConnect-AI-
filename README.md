# Chicago-VetConnect-AI- Version 1
Chicago VetConnect AI is a retrieval-augmented AI assistant designed to help military veterans in Chicago find housing, crisis support and local services. 

The system retrieves information from trusted and verified resource guides and provides responses with citation to reduce hallucinations and misinformation. 

FEATURES: 
- Retrieval Augmented Generation (RAG)
- Crisis phrase detection for emergency situations
- Citations from verified documents
- FastAPI backend
- Groq powered LLM responses

Stack:
- Python
- FastAPI
- Langchain
- Chroma Vector Database
- Groq LLM API

Goal: 
This project is designed to provide reliable access to critical information for veterans experiencing homelessness or are facing housing insecurity in Chicago. 
The assistant draws on collected information to provide information at a person's time of greatest need-the loss of basic human needs. 
It prioritizes accuracy, concise responses and verified sources to reduce the risk of misinformation.
